# Мультиагентная система генерации и проверки гипотез на основе табличных данных

Курсовой проект по программе "Прикладная математика и информатика", НИУ ВШЭ, 2026.

## Описание

Система автоматизирует полный цикл аналитического исследования табличных данных:
от генерации гипотез до формирования структурированного отчёта с интерпретацией результатов на естественном языке.

## Архитектура

Система построена на фреймворке LangGraph и включает шесть агентов:

- **Оркестратор** — анализирует датасет и координирует работу системы
- **Агент генерации гипотез** — формулирует исследовательские гипотезы на основе метаданных
- **Агент выбора метода** — определяет подходящий статистический метод для каждой гипотезы
- **Агент-аналитик** — выполняет вычисления с помощью pandas, scikit-learn, scipy
- **Агент верификации** — проверяет корректность результатов и при необходимости возвращает задачу на пересмотр
- **Агент интерпретации** — формирует текстовые выводы на естественном языке

## Поддерживаемые методы анализа

- Корреляционный анализ (Пирсон, Спирмен)
- Линейная регрессия
- Логистическая регрессия
- Кластеризация (K-Means)
- T-тест
- Описательная статистика
